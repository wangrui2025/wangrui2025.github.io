# 简历引用格式存档

本目录保存 CV 双栏引用格式示例，供后续参考。

## 格式说明

双栏布局：左侧固定 52pt 显示 venue（加粗），右侧显示标题和作者。

## 示例

```
[CVPR 2026] Echocardiography Video Segmentation via Orthogonalized
  State Update and Anatomical Prior-aware Feature Enhancement
  Rui Wang, Huisi Wu*, Jing Qin
```

迁移到单栏布局后，venue 以 `[CVPR 2026]` 标签形式跟在标题后。
