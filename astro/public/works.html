Problem Setting

  • Clinical Goal: Accurate and temporally consistent segmentation of the left ventricle (LV) from echocardiography videos to estimate the ejection fraction ($LV_{EF}$).
  • Imaging Challenges: Severe speckle noise and rapid non-rigid deformations cause blurred anatomical contours across the cardiac cycle.
  • Technical Bottleneck (Rank Collapse): Existing linear recurrent models rely on unconstrained Euclidean state updates. Over long sequences, this causes progressive singular value decay in the state matrix, overwhelming anatomical details with noise.

Method: Orthogonalized State Update (OSU)

To prevent rank collapse, we constrain the memory evolution on the Stiefel Manifold:

$$\mathcal{V}_{C_{v},C_{k}}=\{S\in\mathbb{R}^{C_{v}\times C_{k}}:S^{\top}S=I_{C_{k}}\}$$
$$S_{t}=\mathrm{Proj}_{\mathcal{V}}(S_{t}^{\mathrm{Euc}})$$
  • High-order Newton-Schulz Iteration: We project the unconstrained state back to the manifold efficiently without expensive SVD computation.
  • Geometric Stability: Ensures numerical stability and preserves fine-grained valvular motion across temporal transitions.

Method: Anatomical Prior-aware Feature Enhancement

A physics-driven process explicitly separates anatomical structures from speckle noise:

  • Acoustic Bias Estimation: $M_{t}=AvgPool_{K\times K}(X_{t})$
  • Polarity-aware Decoupling:
    $$X_{t}^{+}=ReLU(X_{t}-M_{t}), \quad X_{t}^{-}=ReLU(M_{t}-X_{t})$$
  • $X_{t}^{+}$ isolates high-frequency structural edges (myocardium boundaries), while $X_{t^{-}$ captures low-response homogeneous regions (blood pools).

Experiments

  • SOTA Performance: Evaluated on CAMUS and EchoNet-Dynamic. OSA achieves 94.82 mDice on CAMUS, surpassing recent vision LRM and state-space models.
  • Real-time Efficiency: Achieves an inference speed of 35 FPS, satisfying the demands of real-time clinical deployment.
  • Geometric Regularization Validated: OSU strictly forces the Singular Value Variance (SVVar) to 0.00 and significantly minimizes Temporal Drift.

Discussion & Code

OSA successfully bridges the gap between temporal tracking stability and ultrasound-domain awareness, providing a reliable foundation for automated $LV_{EF}$ assessment.
