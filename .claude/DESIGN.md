> **用途**: 本文件是网站视觉设计的唯一权威来源。后续所有美术/样式改动必须同步更新此文档。
> **更新日期**: 2026-04-01

---

## 一、设计理念

参考 Claude Docs 的视觉风格，打造**温暖、专业、学术感**的个人主页：

- **暖色调**: 避免冷冰冰的纯白，使用暖灰/米白背景
- **优雅衬线**: 标题使用 Source Serif 4 衬线体，传达书卷气
- **柔和克制**: 扁平化设计、极淡阴影、柔和边框分割
- **呼吸感**: 充足的间距和行高，让内容"呼吸"

## 四、组件样式

### 卡片设计 (PaperCard / Education)

**设计原则**: 凸出感 + 柔和边界

| 属性 | 亮色 | 暗色 | 说明 |
|------|------|------|------|
| 背景 | `#FFFFFF` | `#1E1E1E` | 略浅于页面背景，形成层次 |
| 边框 | 1px solid `#E5E5E5` | 1px solid `#2A2A2A` | 极细的边框 |
| 圆角 | 16px | 16px | 大圆角，现代感 |
| 左边框 | 3px solid accent | 3px solid accent | 强调色的视觉锚点 |
| 阴影 | `0 1px 3px rgba(0,0,0,0.04)` | 同 | 极淡阴影，扁平化 |
| 悬停阴影 | `0 4px 12px rgba(0,0,0,0.08)` | 同 | 悬停时微增 |
| 悬停变换 | `translateY(-1px)` | 同 | 轻柔的上浮 |

### 导航链接 (Masthead)

**设计原则**: 隐形的交互反馈

| 状态 | 样式 |
|------|------|
| 默认 | 无背景色，仅文字 |
| 悬停 | `rgba(0,0,0,0.05)` 浅灰背景 + 0.375rem 圆角 |
| 暗色悬停 | `rgba(255,255,255,0.1)` 半透明白色 |
| 过渡 | 0.2s ease，颜色和背景同时过渡 |

### 侧边栏 (Sidebar)

| 属性 | 值 |
|------|------|
| 宽度 | 25% (`flex: 0 0 25%`) |
| 定位 | Sticky，top: 1rem |
| 圆角 | 12px (`rounded-xl`) |

### 头像

| 属性 | 值 |
|------|------|
| 形状 | 圆形 (`border-radius: 50%`) |
| 尺寸 | 128px × 128px |
| 效果 | 封面裁剪 (`object-fit: cover`) |

## 十、CV 页面设计决策 (2026-04-03)

### 排版术语

1. 从排版学与文档设计角度 (Typography & Document Design)
   - **旁注排版 (Marginalia / Margin Layout)**：将简短的元数据（如会议简称、年份）像"旁注"一样独立放置在左侧边缘区域
   - **悬挂式布局 (Hanging Layout)**：左侧标签作为"锚点"，右侧所有内容在同一条垂直基准线上对齐

2. 从简历设计的通用术语角度 (Resume / CV Design)
   - **两栏式条目布局 (Two-column Entry Layout)**：学术简历标准格式，左栏窄（标签），右栏宽（详情）
   - **时间线布局 (Timeline Layout)**：左侧剥离年份/会议，视觉引导呈垂直时间线

3. 从前端开发与 UI 语义角度 (Web Development & UI)
   - **描述列表布局 (Description List Layout)**：类似 `<dl>`，左侧 [CVPR 2026] 如 `<dt>`，右侧详情如 `<dd>`
   - **键值对布局 (Key-Value Layout)**：左侧 Key，右侧 Value

### 作者列表空格问题
- **问题**：Astro 模板中 author 元素之间有换行，HTML 渲染时会产生多余空格
- **解决**：将 `map` 回调压缩为单行，确保 author span 与 separator span 直接相连
- **代码**：`{pub.authors.map((author, i) => (<><span>...</span>{i < len - 1 && <span>，</span>}</>))}`

### 姓名加粗策略
- **顶部姓名** (`cv-name`)：保留加粗 — 表示页面主人
- **论文作者** (`author-self`)：不加粗 — 与普通作者一致，仅用颜色区分

### 联系方式合并
- **之前**：分两行（电话丨地点）和（生日丨性别）
- **之后**：单行 + 新增 email 和 homepage
- **格式**：`15938550910 丨 深圳 丨 男（he/him）` / `wr_szu@163.com 丨 homepage`

### 荣誉奖项分级
- **主要荣誉**：黑色 (#1a1a1a)
- **次级荣誉**：灰色 (#999)，仅样式区分，不显示"次级"标签
- **数据字段**：`honors[]` 增加 `secondary?: boolean` 标志

### 更新时间格式
- 使用中国时区时间（日期），不使用 "recently"
